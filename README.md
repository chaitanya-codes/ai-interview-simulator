# AI Interview Simulator

An AI-powered interview preparation platform that analyzes resumes, generates personalized interview questions, evaluates responses, and provides actionable feedback to help candidates improve their interview performance.

## Features

### Resume Analysis

* Upload PDF resumes
* Extract candidate information, skills, technologies, projects, and education
* Generate an overall resume score
* Identify strengths and areas for improvement

### AI Interview Generation

* Create interview questions based on resume content
* Generate technical, behavioral, and project-specific questions
* Support multiple difficulty levels

### Response Evaluation

* Score candidate answers
* Provide detailed feedback
* Highlight strengths and weaknesses
* Suggest areas for improvement

### Follow-Up Questions

* Generate contextual follow-up questions
* Simulate realistic interview conversations
* Test depth of knowledge

### Performance Dashboard

* Average interview score
* Question-by-question feedback
* Strength analysis
* Improvement recommendations

## Installation

Clone the repository:

```bash
git clone <repository-url>
cd ai-interview-simulator
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
